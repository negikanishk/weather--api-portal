# SkyNova | Global Weather Intelligence Center 🌍🌩️

> NASA-level global climate monitoring and real-time weather analytics.

SkyNova is an advanced, beautifully designed web application providing comprehensive global weather intelligence. Built with modern web technologies, it features an interactive dashboard that offers real-time climate monitoring, predictive analysis, and an immersive user experience with dynamic visualizations.

## ✨ Features

- **Live Global Monitoring:** Real-time weather analytics including temperature, humidity, wind vector, thermal index, pressure, UV radiation, and visibility.
- **Interactive Globe Visualization:** Experience planetary weather patterns utilizing a custom 3D animated globe powered by Three.js.
- **Predictive Analysis:** 7-Day weather forecasts alongside a graphical Atmospheric Trend Map for deep insights into upcoming climatic shifts.
- **Global Network & Rankings:** Live weather tracking for major global cities, highlighting "Extreme Thermal Zones" and "Global AQI Analysis".
- **Dynamic Search & Geolocation:** Search for climate data by city name, or instantly sync with your current coordinates.
- **Immersive UI/UX:** Glassmorphism aesthetic, sleek animations using Anime.js, and high-quality iconography with Lucide.

## 🛠️ Tech Stack

- **Framework:** Vite
- **DOM & Logic:** Vanilla JavaScript
- **Styling:** CSS (Glassmorphism design, Custom animations)
- **Visualizations:** Chart.js
- **3D Graphics:** Three.js
- **Animations:** Anime.js
- **Icons:** Lucide

## 🚀 Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/weather-api-portal.git
   cd weather-api-portal
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

5. **Preview the production build:**
   ```bash
   npm run preview
   ```

## 📸 Interface Sneak Peek

SkyNova boasts an intricate `Main Dashboard` divided into distinct intelligence cards.
